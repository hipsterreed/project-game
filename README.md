# The Quiet Cliffs

Built for a hackathon powered by [ElevenLabs](https://elevenlabs.io) and [Zed](https://zed.dev).

## The Game

A short atmospheric 3D exploration game. You wake up on a quiet cliff at pre-dawn, find a dormant shrine, and restore it. No HUD, no compass, no lore dumps — the world tells the story. Walk through sand that catches the light, watch flowers bloom at your feet near the shrine, and feel the scene shift as the restoration completes.

Built with Three.js. Spatial audio via ElevenLabs.

## How to Run

```bash
cd game4
npm start
```

Then open [http://localhost:3000](http://localhost:3000).
